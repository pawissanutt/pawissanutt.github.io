# Personal Web Portfolio

Minimal single-page portfolio built with Dioxus 0.7, Tailwind, and WebAssembly.

## Quick Start
- Install Rust toolchain and `dioxus-cli` (`cargo binstall dioxus-cli`).
- Run locally: `dx serve`
- Produce release build: `dx build --platform web --release`

GitHub Actions deploys the release artifacts in `target/dx/personal-web/release/web/public` to GitHub Pages.
